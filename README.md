# README

- System dependencies

  - [Node.js](https://nodejs.org/en/download/) `>= v16.16.0`
    - npm `>= 8.11.0`
  - [k6](https://k6.io/docs/getting-started/installation/) `>= v0.40.0`

- How to run the test suite

  1. `npm i`
  2. `set env vars in the env.js file`
  3. `k6 run scenarios/createTodo.js`
